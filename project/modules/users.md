# Users Module

## Features
- List, create, update, and delete users.
- Assign roles to users.
- View user details.

## Files
- `src/service/users/users.ts`: User API services.
- `src/views/pages/users/`: User management pages (Users, UserForm, UserDetail).
- `src/components/UserRolesManager.vue`: Component for managing user roles.
- `src/router/routes/users/users.routes.ts`: User routes.
