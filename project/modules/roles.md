# Roles Module

## Features
- List, create, update, and delete roles.
- Assign permissions to roles.
- View role details.

## Files
- `src/service/roles/roles.ts`: Role API services.
- `src/service/permissions/permissions.ts`: Permission API services.
- `src/views/pages/roles/`: Role management pages (Roles, RoleForm, RoleDetail).
- `src/components/RolePermissionsManager.vue`: Component for managing role permissions.
- `src/router/routes/roles/roles.routes.ts`: Role routes.
