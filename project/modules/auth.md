# Auth Module

## Features
- User login with email and password.
- JWT-based session management.
- Token storage in `localStorage`.
- Automatic logout on token expiration.

## Files
- `src/service/auth/auth.ts`: Authentication API calls.
- `src/views/pages/auth/Login.vue`: Login page.
- `src/router/routes/auth/auth.routes.ts`: Auth routes.
