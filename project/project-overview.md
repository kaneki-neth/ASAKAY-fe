# Project Overview

## Description
Athlos Frontend is a Vue 3-based administrative dashboard application. It provides a user interface for managing roles, permissions, and users, along with a dashboard for data visualization.

## Goals
- Provide a modern and responsive user interface for administrative tasks.
- Enable efficient management of user roles and permissions.
- Offer data visualization through various dashboard widgets.

## Technology Stack
- **Frontend Framework:** Vue 3 (Composition API)
- **Language:** TypeScript
- **Build Tool:** Vite
- **UI Component Library:** PrimeVue
- **Styling:** Tailwind CSS, PrimeFlex, SASS
- **Routing:** Vue Router
- **HTTP Client:** Axios
- **Charts:** Chart.js
